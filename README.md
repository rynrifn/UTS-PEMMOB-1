Flash News 

Aplikasi Android Native sederhana yang menampilkan portal berita dengan fitur autentikasi (Login & Register). Dibuat menggunakan **Kotlin + XML** untuk memenuhi tugas UTS Pemrograman Mobile 1.

**Fitur Utama**

- Splash Screen
- Halaman Login
- Halaman Register
- Dashboard Berita

**Teknologi**

- Kotlin
- Android Studio
- XML Layout
